# Gaurav ♥ Suhani — Wedding Website

Save-the-date / wedding website for Gaurav & Suhani's celebration on November 21, 2026 in Rampur, Uttar Pradesh.

## Structure

Single self-contained static page — no build step required.

- `index.html` — the entire site (HTML, CSS, and JS inline)

## Deploying

This is a static site, so it deploys as-is on Vercel, Netlify, GitHub Pages, or any static host — no framework or build command needed.

### Vercel
1. Import this repository at https://vercel.com/new
2. Leave the framework preset as "Other" and the build command empty
3. Deploy — Vercel will serve `index.html` directly

Every push to the connected branch will trigger a new deployment automatically.
